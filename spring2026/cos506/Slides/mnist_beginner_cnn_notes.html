Beginner CNN for MNIST Classification

This page explains a beginner version of a Convolutional Neural Network (CNN) for MNIST. A CNN is often better than a simple dense network for images because it can learn local patterns such as edges, curves, and strokes.

Why use a CNN?

Reason 1

A CNN keeps the 2D image structure instead of flattening immediately.

Reason 2

It learns small visual patterns such as corners, lines, and curves.

Reason 3

It usually gives better accuracy on image tasks than a basic dense model.

1. Load and normalize the dataset

We start with the same MNIST images, but this time we will keep the image shape because a CNN works directly on image grids. 

(x_train, y_train), (x_test, y_test) = keras.datasets.mnist.load_data()

x_train = x_train.astype("float32") / 255.0
x_test = x_test.astype("float32") / 255.0

2. Reshape the images for CNN input

CNN layers expect height, width, and channels. MNIST images are grayscale, so the number of channels is 1. 

x_train = x_train.reshape((-1, 28, 28, 1))
x_test = x_test.reshape((-1, 28, 28, 1))
Meaning of 28 × 28 × 1:
28 = image height
28 = image width
1 = one grayscale channel

3. Build a small CNN model

This beginner version keeps the structure simple: one convolution layer, one pooling layer, then flatten and dense layers. 

model = keras.Sequential([
    keras.Input(shape=(28, 28, 1)),
    layers.Conv2D(16, (3, 3), activation="relu"),
    layers.MaxPooling2D((2, 2)),
    layers.Flatten(),
    layers.Dense(64, activation="relu"),
    layers.Dense(10, activation="softmax")
])
Layer What it does
Conv2D(16, (3, 3)) Looks at small 3×3 parts of the image and learns 16 useful filters.
MaxPooling2D((2, 2)) Makes the feature maps smaller and keeps important information.
Flatten() Turns the 2D feature maps into a 1D vector for the dense layer.
Dense(64, activation="relu") Learns higher-level combinations of the extracted features.
Dense(10, activation="softmax") Outputs probabilities for the 10 digit classes.

4. What is convolution?

Convolution means sliding a small filter over the image to detect simple patterns. Early filters often learn shapes like edges, curves, and stroke pieces.

Beginner picture idea: think of a tiny window moving across the image and checking what pattern is present in each area.

5. What is max pooling?

Max pooling reduces the size of the feature map. It keeps the strongest signal in each small region. This helps the model focus on important patterns and reduces computation.

For example, a 2×2 pooling window looks at four nearby values and keeps only the largest one.

6. Compile the model

The compile step is almost the same as in the dense classifier because this is still a 10-class classification problem. 

model.compile(
    optimizer="adam",
    loss="sparse_categorical_crossentropy",
    metrics=["accuracy"]
)

7. Train the CNN

The model learns by seeing many examples of labeled digit images. 

history = model.fit(
    x_train, y_train,
    validation_split=0.1,
    epochs=5,
    batch_size=128,
    verbose=1
)
A beginner CNN often performs very well on MNIST even with a small number of epochs.

8. Evaluate and predict

After training, we evaluate on test data and then predict labels for some unseen images. 

test_loss, test_accuracy = model.evaluate(x_test, y_test, verbose=0)
print("Test Accuracy:", test_accuracy)

predictions = model.predict(x_test[:10], verbose=0)
predicted_labels = np.argmax(predictions, axis=1)

Complete beginner CNN code

import numpy as np
import matplotlib.pyplot as plt
from tensorflow import keras
from tensorflow.keras import layers

# 1) Load data
(x_train, y_train), (x_test, y_test) = keras.datasets.mnist.load_data()

# 2) Normalize
x_train = x_train.astype("float32") / 255.0
x_test = x_test.astype("float32") / 255.0

# 3) Reshape for CNN
x_train = x_train.reshape((-1, 28, 28, 1))
x_test = x_test.reshape((-1, 28, 28, 1))

# 4) Build beginner CNN
model = keras.Sequential([
    keras.Input(shape=(28, 28, 1)),
    layers.Conv2D(16, (3, 3), activation="relu"),
    layers.MaxPooling2D((2, 2)),
    layers.Flatten(),
    layers.Dense(64, activation="relu"),
    layers.Dense(10, activation="softmax")
])

# 5) Compile
model.compile(
    optimizer="adam",
    loss="sparse_categorical_crossentropy",
    metrics=["accuracy"]
)

# 6) Train
history = model.fit(
    x_train, y_train,
    validation_split=0.1,
    epochs=5,
    batch_size=128,
    verbose=1
)

# 7) Evaluate
test_loss, test_accuracy = model.evaluate(x_test, y_test, verbose=0)
print("Test Accuracy:", test_accuracy)

# 8) Predict
predictions = model.predict(x_test[:10], verbose=0)
predicted_labels = np.argmax(predictions, axis=1)

# 9) Show some predictions
plt.figure(figsize=(14, 4))
for i in range(10):
    plt.subplot(2, 5, i + 1)
    plt.imshow(x_test[i].reshape(28, 28), cmap="gray")
    plt.title(f"Pred: {predicted_labels[i]}\\nTrue: {y_test[i]}")
    plt.axis("off")

plt.tight_layout()
plt.show()

Dense model vs CNN

Dense classifier CNN classifier
Flattens image into 784 values at the beginning Keeps the image structure longer
Very simple to teach first Better for teaching image-specific learning
Usually lower accuracy than CNN on images Usually higher accuracy on images

Short teaching summary

A beginner CNN is a natural next step after a dense classifier. It helps understand that image tasks are special: the arrangement of pixels matters. With just one convolution layer and one pooling layer, you can already see why CNNs are powerful.
