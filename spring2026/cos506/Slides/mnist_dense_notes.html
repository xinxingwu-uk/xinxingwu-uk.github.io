MNIST Classification with a Simple Dense Neural Network

This page explains a beginner-friendly classification example based on MNIST. The goal is to learn how a neural network can look at a handwritten digit image and decide whether it is 0, 1, 2, ..., or 9.

Big Idea

Classification means choosing a label from several possible classes.

In this example, the classes are the digits 0 through 9.

Input

Each image is 28 × 28 pixels.

That gives 784 values after flattening.

Output

The final layer has 10 neurons.

Each neuron gives the probability of one digit class.

1. Load the dataset

MNIST is a very popular beginner dataset. It contains many small grayscale images of handwritten digits. It is a good starting point because it is simple, clean, and fast to train. 

(x_train, y_train), (x_test, y_test) = keras.datasets.mnist.load_data()
What are these variables?
x_train = training images
y_train = correct labels for training images
x_test = test images
y_test = correct labels for test images

2. Normalize the pixel values

Pixel values in MNIST go from 0 to 255. Dividing by 255 changes them to the range 0 to 1. This makes training easier and usually helps the model learn better. 

x_train = x_train.astype("float32") / 255.0
x_test = x_test.astype("float32") / 255.0
Simple explanation: smaller numbers are easier for the neural network to work with.

3. Flatten each image

A dense neural network expects a one-dimensional input vector. Since each image is 28 × 28, we convert it into a row of 784 values. 

x_train = x_train.reshape((-1, 28 * 28))
x_test = x_test.reshape((-1, 28 * 28))

The -1 tells Python to keep the number of images automatically.

4. Build the model

This model has two hidden layers and one output layer. The hidden layers use ReLU, and the output layer uses softmax because this is a multi-class classification problem. 

model = keras.Sequential([
    keras.Input(shape=(784,)),
    layers.Dense(128, activation="relu"),
    layers.Dense(64, activation="relu"),
    layers.Dense(10, activation="softmax")
])
Part Meaning
Input(shape=(784,)) The model receives 784 numbers for each image.
Dense(128, activation="relu") First hidden layer learns useful patterns from the pixels.
Dense(64, activation="relu") Second hidden layer builds more abstract features.
Dense(10, activation="softmax") Outputs probabilities for the 10 digit classes.
Why softmax?
Softmax turns the final outputs into probabilities that add up to 1. The largest probability becomes the prediction.

5. Compile the model

Compiling tells Keras how to train the model. We choose an optimizer, a loss function, and a metric. 

model.compile(
    optimizer="adam",
    loss="sparse_categorical_crossentropy",
    metrics=["accuracy"]
)
Term Beginner meaning
adam A popular optimizer that updates weights efficiently.
sparse_categorical_crossentropy Good for multi-class classification when labels are integers like 0, 1, 2, ..., 9.
accuracy Measures the percentage of correct predictions.

6. Train the model

Training means showing many labeled examples to the neural network so it can gradually improve. 

history = model.fit(
    x_train, y_train,
    validation_split=0.1,
    epochs=10,
    batch_size=256,
    verbose=1
)

During training, Keras reports values such as training accuracy and validation accuracy. You can observe whether the model is learning better from one epoch to the next.

Key terms:
Epoch = one full pass through the training data
Batch size = number of samples processed at one time
Validation split = part of the training data used to check progress

7. Evaluate the model

After training, we test the model on data it did not use during learning. This gives a more honest measure of performance. 

test_loss, test_accuracy = model.evaluate(x_test, y_test, verbose=0)
print("Test Loss:", test_loss)
print("Test Accuracy:", test_accuracy)

If the test accuracy is high, the model is doing well on unseen examples.

8. Make predictions

The model produces a probability for each class. We usually choose the class with the highest probability. 

predictions = model.predict(x_test[:10], verbose=0)
predicted_labels = np.argmax(predictions, axis=1)
Example: if the model outputs probabilities and the largest one is at index 7, then the predicted digit is 7.

9. Visualize the results

It is helpful for beginners to see the image together with the predicted label and the true label. This makes the model behavior easier to understand. 

plt.imshow(x_test[i].reshape(28, 28), cmap="gray")
plt.title(f"Pred: {predicted_labels[i]}\nTrue: {y_test[i]}")

You can immediately notice which digits were predicted correctly and which ones were difficult.

10. Why this example is beginner-friendly

This example is suitable for beginners because the data is easy to understand, the model is short, and the final output is intuitive. You can see a digit and understand why the answer should be one of 10 classes.

Good for beginners because... Explanation
Simple dataset Everyone understands handwritten digits.
Small code size The code is short enough to explain in one class session.
Clear output The model predicts 0–9, which is easy to interpret.
Easy extension You can later compare this model with a CNN.

Complete beginner version of the Python code

import numpy as np
import matplotlib.pyplot as plt
from tensorflow import keras
from tensorflow.keras import layers

# 1) Load data
(x_train, y_train), (x_test, y_test) = keras.datasets.mnist.load_data()

# 2) Normalize
x_train = x_train.astype("float32") / 255.0
x_test = x_test.astype("float32") / 255.0

# 3) Flatten images
x_train = x_train.reshape((-1, 28 * 28))
x_test = x_test.reshape((-1, 28 * 28))

# 4) Build model
model = keras.Sequential([
    keras.Input(shape=(784,)),
    layers.Dense(128, activation="relu"),
    layers.Dense(64, activation="relu"),
    layers.Dense(10, activation="softmax")
])

# 5) Compile model
model.compile(
    optimizer="adam",
    loss="sparse_categorical_crossentropy",
    metrics=["accuracy"]
)

# 6) Train model
history = model.fit(
    x_train, y_train,
    validation_split=0.1,
    epochs=10,
    batch_size=256,
    verbose=1
)

# 7) Evaluate model
test_loss, test_accuracy = model.evaluate(x_test, y_test, verbose=0)
print("Test Accuracy:", test_accuracy)

# 8) Predict first 10 images
predictions = model.predict(x_test[:10], verbose=0)
predicted_labels = np.argmax(predictions, axis=1)

# 9) Show results
plt.figure(figsize=(14, 4))
for i in range(10):
    plt.subplot(2, 5, i + 1)
    plt.imshow(x_test[i].reshape(28, 28), cmap="gray")
    plt.title(f"Pred: {predicted_labels[i]}\\nTrue: {y_test[i]}")
    plt.axis("off")

plt.tight_layout()
plt.show()

Short teaching summary

A dense classifier for MNIST is one of the best first examples for deep learning. It is how to prepare data, build a model, train it, test it, and visualize predictions. It also prepares for the next step: using a CNN, which usually performs better on images.
