MNIST Digit Classification - Beginner Dense Model

This page is a beginner browser version of the Python example. It uses TensorFlow.js instead of Python TensorFlow.
Goal: look at a handwritten digit image and classify it as 0 to 9.

What this model does

  • Loads MNIST handwritten digit images.
  • Flattens each 28x28 image into 784 numbers.
  • Uses a simple dense neural network.
  • Predicts one of 10 digit classes.
